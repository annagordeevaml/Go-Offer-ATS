# Проверка URL Supabase проекта

## Проблема: ERR_NAME_NOT_RESOLVED

Ошибка означает, что домен `hyoqcxaielpvmmluanogv.supabase.co` не может быть найден.

## Как проверить правильный URL:

1. Откройте Supabase Dashboard: https://app.supabase.com
2. Войдите в свой аккаунт
3. Выберите ваш проект
4. Перейдите в **Settings** → **API**
5. Найдите **Project URL** - это правильный URL вашего проекта
6. Он должен выглядеть как: `https://xxxxxxxxxxxxx.supabase.co`

## Как обновить URL:

1. Скопируйте правильный **Project URL** из Supabase Dashboard
2. Откройте файл `.env.local`
3. Обновите строку:
   ```
   VITE_SUPABASE_URL=https://ваш_правильный_url.supabase.co
   ```
4. Сохраните файл
5. Перезапустите dev-сервер: `npm run dev`

## Возможные причины проблемы:

1. **Проект был удален или приостановлен** - проверьте статус проекта в Dashboard
2. **Неправильный URL** - убедитесь, что используете URL из Settings → API
3. **Проблемы с интернетом** - проверьте подключение к интернету
4. **DNS проблемы** - попробуйте использовать другой DNS (например, 8.8.8.8)

## Проверка доступности:

После обновления URL, проверьте в терминале:
```bash
curl -I https://ваш_url.supabase.co
```

Если получаете ответ с HTTP статусом (200, 301, и т.д.) - URL правильный.
Если получаете "Could not resolve host" - проверьте URL в Dashboard.
